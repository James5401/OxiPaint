## ✦ Features

| | |
|---|---|
| **23 Drawing Tools** | Select, Hand, Pencil, Highlighter, Eraser, Fill, Rectangle, Ellipse, Triangle, Diamond, Star, Hexagon, Callout, Arrow, Line, Connector, Text, Sticky Note, Image, Laser, Frame |
| **Rich Styling** | 4 sizes · solid/dashed/dotted dashes · opacity slider · 3 fill modes + linear & radial gradients |
| **13 Stroke Effects** | Glow, Neon, Chalk, Shadow, Blur, Rough, Outline, Fire, Ice, Emboss, Glitter, Vintage |
| **10 Pattern Brushes** | Dots, Rings, Stars, Diamonds, Arrows, Zigzag, Crosses, Hearts, Spray — plus custom brush creator |
| **10 Base Colors + Custom** | Stroke, fill, and highlight color pickers with auto-saved custom palettes |
| **Multi-Page** | Add, delete, and navigate between canvases — each with its own drawings |
| **Layers & Groups** | Bring to front / send to back · lock shapes · group & ungroup strokes |
| **Undo / Redo** | Full history stack (80 snapshots) — <kbd>Ctrl+Z</kbd> / <kbd>Ctrl+Y</kbd> |
| **Copy / Paste / Duplicate** | Internal clipboard for shapes + paste images from OS clipboard |
| **Image Support** | Upload, paste, crop, and apply 10 image effect presets with individual adjustments |
| **Connector Tool** | Curve / Straight / Elbow lines with arrowheads, dots, diamonds, and labels |
| **Text Tool** | Sans / Serif / Mono / Script fonts · bold / italic / underline · alignment · line height |
| **Sticky Notes** | Colorful notes with auto-sized editable text |
| **Eraser** | Object mode (delete whole shapes) or Pixel mode (erase stroke portions) |
| **Fill Bucket** | Click-to-fill shapes or flood-fill canvas areas with adjustable tolerance |
| **Laser Pointer** | Red comet-trail pointer perfect for presentations |
| **Pan & Zoom** | 5%–800% zoom · mousewheel · two-finger pinch · fit-to-content |
| **Snap to Grid** | Toggle with <kbd>G</kbd> — 20px snapping for precise placement |
| **Board Background** | 10 solid colors + custom picker · 10 patterns (dots, lines, grid, isometric, circles, checker, etc.) |
| **Apple Design System UI** | Native macOS/iOS aesthetics · frosted glass panels · light & dark mode |
| **Export PNG** | 2x retina export via <kbd>Ctrl+S</kbd> or toolbar button |
| **Persistence** | Everything auto-saves to `localStorage` — drawings survive page reloads |

## ✦ Keyboard Shortcuts

### Toggle
| Shortcut | Action |
|---|---|
| <kbd>Alt+W</kbd> | Open / close whiteboard |

### Tool Selection
| Key | Tool | Key | Tool |
|---|---|---|---|
| <kbd>V</kbd> | Select | <kbd>B</kbd> | Fill Bucket |
| <kbd>H</kbd> | Hand (Pan) | <kbd>R</kbd> | Rectangle |
| <kbd>P</kbd> | Pencil | <kbd>O</kbd> | Ellipse |
| <kbd>I</kbd> | Highlighter | <kbd>A</kbd> | Arrow |
| <kbd>E</kbd> | Eraser | <kbd>L</kbd> | Line |
| <kbd>F</kbd> | Frame | <kbd>C</kbd> | Connector |
| <kbd>T</kbd> | Text | <kbd>S</kbd> | Sticky Note |
| <kbd>G</kbd> | Snap to Grid | | |

### Editing
| Shortcut | Action |
|---|---|
| <kbd>Ctrl+Z</kbd> | Undo |
| <kbd>Ctrl+Y</kbd> / <kbd>Ctrl+Shift+Z</kbd> | Redo |
| <kbd>Ctrl+C</kbd> | Copy |
| <kbd>Ctrl+X</kbd> | Cut |
| <kbd>Ctrl+V</kbd> | Paste |
| <kbd>Ctrl+D</kbd> | Duplicate |
| <kbd>Ctrl+A</kbd> | Select All |
| <kbd>Ctrl+S</kbd> | Export PNG |
| <kbd>Delete</kbd> / <kbd>Backspace</kbd> | Delete |
| <kbd>Escape</kbd> | Deselect / Select tool |
| <kbd>↑</kbd> <kbd>↓</kbd> <kbd>←</kbd> <kbd>→</kbd> | Nudge 1px (+ <kbd>Shift</kbd> for 10px) |

### Text Formatting (inline)
| Shortcut | Action |
|---|---|
| <kbd>Ctrl+B</kbd> | Bold |
| <kbd>Ctrl+I</kbd> | Italic |
| <kbd>Ctrl+U</kbd> | Underline |

## ✦ Installation

1. Install a userscript manager like **[Violentmonkey](https://violentmonkey.github.io/)** or **[Tampermonkey](https://www.tampermonkey.net/)**.
2. Open [`OxiPaint (Whiteboard) 6.2.user.js`](./OxiPaint%20(Whiteboard)%206.2.user.js) and click **Install**.
3. Press <kbd>Alt+W</kbd> on any page to start drawing.
